Petunjuk Pengunggahan File Audio Lokal:

1. Letakkan file audio Anda (.mp3, .m4a, .wav, .ogg, dll.) ke dalam folder ini (public/audio/).
2. Di editor DocaCMS, klik tombol "Audio" di toolbar.
3. Masukkan path URL relatif file Anda, contohnya:
   /audio/nama_file_anda.mp3

File tersebut akan langsung terbaca dan dapat diputar di tempat (inline) di dalam halaman dokumentasi Anda, baik di mode editor maupun tampilan publik (reader view).
